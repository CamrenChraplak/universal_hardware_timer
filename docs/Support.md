# Support

Listed below are supported development environments and supported boards. Some boards developed from outside entities that use these chips are mentioned. Others, however, will need to be searched by the user.

## Supported IDEs:

- [Arduino IDE](https://www.arduino.cc/en/software/) (All supported boards)
- [ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html) (All supported Espressif ESP32 boards)
- [Raspberry Pi Pico SDK](https://www.raspberrypi.com/documentation/microcontrollers/c_sdk.html) (All supported Raspberry Pi Pico boards)
- [PlatformIO](https://platformio.org/) (All supported boards)
